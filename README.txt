# Vješala — GitHub Pages paket
Sadržaj: index.html, manifest.webmanifest, service-worker.js, icons/
## Deploy
1. Kreiraj public repo (npr. `vjesala`).
2. Uploadaj SVE fajlove iz ZIP-a u root.
3. Settings → Pages → Source: Deploy from a branch → Branch: main → Folder: / (root) → Save.
4. URL: https://<tvoj-username>.github.io/<repo>/
